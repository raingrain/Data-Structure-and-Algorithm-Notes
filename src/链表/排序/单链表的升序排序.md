## 程序员代码面试指南：IT名企算法与数据结构题目最优解（第二版） 第二章 链表问题 单链表的选择排序

> - ***Question***
>   - 给定单链表的头节点 `head` ，请使用选择排序、插入排序、归并排序和快速排序对单链表进行升序排序，然后返回排好序的单链表的头节点。
>   - ***tips:***
>     - 链表中节点的数目在范围 `[0, 5 * 10^4]` 内
>     - `-10^5 <= Node.val <= 10^5`

---

## *Java*

> - ***选择排序***
>   - 时间复杂度为 `O(N ^ 2)` ，空间复杂度为 `O(1)` 。
> - ***插入排序***
>   - 时间复杂度为 `O(N ^ 2)` ，空间复杂度为 `O(1)` 。
> - ***自顶向下归并排序***
>   - 时间复杂度为 `O(N * logN)` ，空间复杂度为 `O(logN)` ，空间复杂度主要取决于递归调用的栈空间。
> - ***自底向上归并排序***
>   - 时间复杂度为 `O(N * logN)` ，空间复杂度为 `O(1)` 。
> - ***快速排序***
>   - 时间复杂度为 `O(N * logN)` ，空间复杂度为 `O(logN)` 。

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 * int val;
 * ListNode next;
 * ListNode() {}
 * ListNode(int val) { this.val = val; }
 * ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    
    public ListNode sortList(ListNode head) {
        
        return quickSort(head);
        // return mergeSortFromEndToTop(head);
        // return mergeSortFromTopToEnd(head, null);
        // return insertionSortList(head);
        // return selectionSort(head);
    }
    
    // 快速排序
    public ListNode quickSort(ListNode head) {
        if (head == null || head.next == null) {
            // 递归的终止条件，链表长度小于1
            return head;
        } else {
            // 这些头也是哨兵守卫
            // 按顺序分别是小于区、等于区和大于区
            ListNode head1 = new ListNode();
            ListNode head2 = new ListNode();
            ListNode head3 = new ListNode();
            ListNode tail1 = head1;
            ListNode tail2 = head2;
            ListNode tail3 = head3;
            // 遍历用的
            ListNode cur = head;
            // 分区基准
            // 找到快速排序的基准
            // 因为不知道链表长度，找到长度还需要遍历一次链表，所以不随机选择基准
            // 直接选择用中点作为基准
            int pivot = getMid(head, null).val;
            
            while (cur != null) {
                // 先抓住cur的下一个
                ListNode next = cur.next;
                // 断开连接
                cur.next = null;
                // 根据值的大小分到不同的区，并更新区尾
                if (cur.val < pivot) {
                    tail1.next = cur;
                    tail1 = tail1.next;
                } else if (cur.val == pivot) {
                    tail2.next = cur;
                    tail2 = tail2.next;
                } else {
                    tail3.next = cur;
                    tail3 = tail3.next;
                }
                // cur移动
                cur = next;
            }
            
            // 等于区有序，只需要对大于小于区排序
            head1 = quickSort(head1.next);
            head3 = quickSort(head3.next);
            
            // head2是哨兵守卫，且等于区一定不为空
            head2 = head2.next;
            
            // 不管head3为不为空都可以这样写
            tail2.next = head3;
            
            // 如果head1为空直接返回等于区及以后
            if (head1 == null) {
                return head2;
            } else {
                // 否则找到小于区的尾巴，将其指向等于区
                cur = head1;
                while (cur.next != null) {
                    cur = cur.next;
                }
                cur.next = head2;
                // 返回小于区
                return head1;
            }
        }
    }
    
    // 找到一段链表的重点，tail是最后一个节点的next
    public ListNode getMid(ListNode head, ListNode tail) {
        ListNode fast = head;
        ListNode slow = head;
        while (fast != tail && fast.next != tail) {
            fast = fast.next.next;
            slow = slow.next;
        }
        return slow;
    }
    
    // 自底向上归并排序
    // 首先求得链表的长度length 然后将链表拆分成子链表进行合并
    // 用subLength表示每次需要排序的子链表的长度，初始时subLength=1
    // 每次将链表拆分成若干个长度为subLength的子链表（最后一个子链表的长度可以小于subLength）
    // 按照每两个子链表一组进行合并，合并后即可得到若干个长度为subLength×2的有序子链表（最后一个子链表的长度可以小于subLength×2）
    // 将subLength的值加倍，重复对更长的有序子链表进行合并操作
    // 直到有序子链表的长度大于或等于length，整个链表排序完毕
    public ListNode mergeSortFromEndToTop(ListNode head) {
        if (head == null) {
            return null;
        } else {
            // 获取链表长度
            int length = 0;
            ListNode cur = head;
            while (cur != null) {
                length++;
                cur = cur.next;
            }
            ListNode dummy = new ListNode(0, head);
            ListNode pre;
            // 直到合成的链表长度大于等于原长度，每次乘2
            for (int subLength = 1; subLength < length; subLength <<= 1) {
                // 这一次长度为subLength中排好序的链表的最后一个
                // 即每一轮都要重新构建以dummy为头的链表
                pre = dummy;
                cur = dummy.next;
                // 遍历一遍链表
                while (cur != null) {
                    // 第一组长度为subLength的子链表
                    ListNode head1 = cur;
                    // 可能还没凑够subLength链表就结束了
                    for (int i = 1; i < subLength && cur.next != null; ++i) {
                        cur = cur.next;
                    }
                    // 第二组长度为subLength的子链表
                    ListNode head2 = cur.next;
                    // 第一组链表先从原链表中删除
                    cur.next = null;
                    cur = head2;
                    // 可能只有第一组或者第二组没走完链表就结束了
                    for (int i = 1; i < subLength && cur != null && cur.next != null; ++i) {
                        cur = cur.next;
                    }
                    ListNode next = null;
                    // cur为空证明第二组的长度为subLength
                    if (cur != null) {
                        // 记住下一个cur该去的节点
                        next = cur.next;
                        // 将第二组从原链表中删除
                        cur.next = null;
                    }
                    // 接到dummy中
                    pre.next = merge(head1, head2);
                    // pre移动到已经排好序的链表的最后一个
                    while (pre.next != null) {
                        pre = pre.next;
                    }
                    // 去往下一个要排序的区域
                    cur = next;
                }
            }
            return dummy.next;
        }
    }
    
    // 自顶向下的归并排序，递归版
    // 找到链表的中点，以中点为分界，将链表拆分成两个子链表
    // 寻找链表的中点可以使用快慢指针的做法
    // 对两个子链表分别排序
    // 将两个排序后的子链表合并，得到完整的排序后的链表
    public ListNode mergeSortFromTopToEnd(ListNode head, ListNode tail) {
        // 链表为空直接返回空
        if (head == null) {
            return null;
        } else if (head.next == tail) {
            // 如果链表只有一个节点，直接将其从原链表断开并返回这个节点
            head.next = null;
            return head;
        } else {
            // 找到中点
            ListNode mid = getMid(head, tail);
            // 左右区间去递归
            ListNode list1 = mergeSortFromTopToEnd(head, mid);
            ListNode list2 = mergeSortFromTopToEnd(mid, tail);
            // 合并两个有序链表
            return merge(list1, list2);
        }
    }
    
    // 合并两个链表
    public ListNode merge(ListNode head1, ListNode head2) {
        ListNode dummy = new ListNode(0);
        ListNode pre = dummy, cur1 = head1, cur2 = head2;
        while (cur1 != null && cur2 != null) {
            if (cur1.val <= cur2.val) {
                pre.next = cur1;
                cur1 = cur1.next;
            } else {
                pre.next = cur2;
                cur2 = cur2.next;
            }
            pre = pre.next;
        }
        if (cur1 != null) {
            pre.next = cur1;
        } else if (cur2 != null) {
            pre.next = cur2;
        }
        return dummy.next;
    }
    
    // 对链表进行插入排序
    // 首先判断给定的链表是否为空，若为空，则不需要进行排序，直接返回
    // 创建哑节点dummy，令dummy.next=head
    // 引入哑节点是为了便于在 head 节点之前插入节点
    // 维护last为链表的已排序部分的最后一个节点，初始时last为head
    // 维护cur为待插入的元素，初始时cur=head.next
    // 比较last和cur的节点值
    // 若last.val<=cur.val，说明cur应该位于last之后，将last后移一位，cur变成新的last
    // 否则，从链表的头节点开始往后遍历链表中的节点，寻找插入cur的位置并将其插入，同时与原链表断开
    // 重复直到cur变成空，排序结束，返回排序后的链表的头节点dummy.next
    public ListNode insertionSortList(ListNode head) {
        if (head == null) {
            return null;
        } else {
            ListNode dummy = new ListNode(0, head);
            // last指已经排序的链表中的最后一个节点，cur是当前待排序的节点
            ListNode last = head, cur = head.next;
            while (cur != null) {
                if (last.val <= cur.val) {
                    // 比已经排序的链表的最大值还要大，直接尾插至已排序链表
                    last = cur;
                } else {
                    // 在已排序链表中找到待插入位置的前一个节点
                    ListNode pre = dummy;
                    while (pre.next.val <= cur.val) {
                        pre = pre.next;
                    }
                    // 把cur从未排序链表中删除
                    last.next = cur.next;
                    // 插入已排序链表
                    cur.next = pre.next;
                    pre.next = cur;
                }
                // 已排序区的下一个一定是未排序
                cur = last.next;
            }
            return dummy.next;
        }
    }
    
    // 对链表进行选择排序
    // 开始时默认整个链表都是未排序的部分，对于找到的第一个最小值节点，肯定是整个链表的最小值节点，将其设置为新链表的头节点，记为哨兵守卫newHead的next
    // 每次在未排序的部分中找到最小值的节点，然后把这个节点从未排序的链表中删除
    // 把删除的节点（也就是每次的最小值节点）连接到排好序部分的链表尾部
    // 全部过程处理完后，整个链表都已经有序，返回newHead.next
    public ListNode selectionSort(ListNode head) {
        // 已经排好序的新链表的头
        ListNode newHead = new ListNode();
        // 已经排好序的新链表的尾
        ListNode tail = newHead;
        // 当前正在检查的元素
        ListNode cur = head;
        // 最小元素的上一个元素
        ListNode smallPre = null;
        // 最小元素
        ListNode small = null;
        while (cur != null) {
            // 获得以cur为头的未排序链表中的最小值的上一个元素
            smallPre = getSmallPreListNode(cur);
            // 如果为空则说明最小元素是第一个
            if (smallPre != null) {
                // 最小元素就是smallPre的下一个
                small = smallPre.next;
                // 把small从未排序链表中断开
                smallPre.next = small.next;
            } else {
                // cur是未排序链表的头
                small = cur;
            }
            // 如果未排序链表的头是最小元素，那cur后移，否则cur不动
            cur = cur == small ? cur.next : cur;
            // 尾插至排好序的链表中
            // 使用哨兵节点避免分类讨论
            tail.next = small;
            tail = small;
        }
        return newHead.next;
    }
    
    // 在一个链表中找到值最小的节点的前一个节点，如果第一个节点就是最小的就返回空
    public ListNode getSmallPreListNode(ListNode head) {
        // 记录最小节点的前一个节点
        ListNode smallPre = null;
        // 记录最小节点
        ListNode small = head;
        // 两个遍历节点
        ListNode pre = head;
        ListNode cur = head.next;
        while (cur != null) {
            // 出现过多次以第一次为准
            if (cur.val < small.val) {
                smallPre = pre;
                small = cur;
            }
            pre = cur;
            cur = cur.next;
        }
        return smallPre;
    }
    
}
```
