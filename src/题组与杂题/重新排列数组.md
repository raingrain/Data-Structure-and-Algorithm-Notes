# 重新排列数组

## [1470. 重新排列数组](https://leetcode.cn/problems/shuffle-the-array/)

> - ***Question***
>   - 给你一个数组 `nums` ，数组中有 `2n` 个元素，按 `[x1, x2, ..., xn, y1, y2, ..., yn]` 的格式排列。请你将数组按 `[x1, y1, x2, y2, ..., xn, yn]` 格式重新排列，返回重排后的数组。
>   - ***tips:***
>     - `1 <= n <= 500`
>     - `nums.length == 2n`
>     - `1 <= nums[i] <= 10^3`

## Java

> - ***一次遍历***

```java
class Solution {

    public int[] shuffle(int[] nums, int n) {
        int[] ans = new int[2 * n];
        for (int i = 0; i < n; i++) {
            // 位置关系
            ans[2 * i] = nums[i];
            ans[2 * i + 1] = nums[i + n];
        }
        return ans;
    }

}
```
