# 让无重复数字无序数组变成有序数组的最小交换次数

## 小红书

> - ***Question***
>   - 一个无序数组长度为 `n` ，所有数字都不一样，并且值都在 `[0...n - 1]` 范围上。返回让这个无序数组变成有序数组的最小交换次数。

---

## *Java*

> - ***下标循环怼***

```java
class Solution {
    
    public static int minSwap(int[] arr) {
        int ans = 0;
        // 让arr[i] = i
        for (int i = 0; i < arr.length; i++) {
            // 交换完后继续看当前位置的数字
            while (i != arr[i]) {
                swap(arr, i, arr[i]);
                // 交换一次答案加一次
                ans++;
            }
        }
        return ans;
    }
    
    private static void swap(int[] arr, int i, int j) {
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }
    
}
```
