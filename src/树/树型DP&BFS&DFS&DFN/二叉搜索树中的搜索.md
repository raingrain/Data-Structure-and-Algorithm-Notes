# 二叉搜索树中的搜索

## [700. 二叉搜索树中的搜索](https://leetcode.cn/problems/search-in-a-binary-search-tree/)

> - ***Question***
>   - 给定二叉搜索树的根节点 `root` 和一个整数值 `val` 。
>   - 你需要在 `BST` 中找到节点值等于 `val` 的节点。 返回以该节点为根的子树。如果节点不存在，则返回 `null` 。
>   - ***tips:***
>     - 树中节点数在 `[1, 5000]` 范围内
>     - `1 <= Node.val <= 10^7`
>     - `root` 是二叉搜索树
>     - `1 <= val <= 10^7`

---

## *Java*

> - ***递归***

```java
class TreeNode {
    
    int val;
    TreeNode left;
    TreeNode right;
    
    TreeNode() {}
    
    TreeNode(int val) {this.val = val;}
    
    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
    
}

class Solution {

    public TreeNode searchBST(TreeNode root, int val) {
        if (root == null) {
            return null;
        }
        if (val == root.val) {
            return root;
        }
        return searchBST(val < root.val ? root.left : root.right, val);
    }

}
```

> - ***迭代***

```java
class TreeNode {

    int val;
    TreeNode left;
    TreeNode right;

    TreeNode() {
    }

    TreeNode(int val) {
        this.val = val;
    }

    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }

}

class Solution {

    public TreeNode searchBST(TreeNode root, int val) {
        while (root != null) {
            if (val == root.val) {
                return root;
            }
            root = val < root.val ? root.left : root.right;
        }
        return null;
    }

}
```
