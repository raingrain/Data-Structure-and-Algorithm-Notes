# 重复的子字符串

## [459. 重复的子字符串](https://leetcode.cn/problems/repeated-substring-pattern/)

> - ***Question***
>   - 给定一个非空的字符串 `s` ，检查是否可以通过由它的一个子串重复多次构成。
>   - ***tips:***
>     - `1 <= s.length <= 10^4`
>     - `s` 由小写英文字母组成

## Java

> - ***API***

```java
class Solution {

    public boolean repeatedSubstringPattern(String s) {
        return (s + s).indexOf(s, 1) != s.length();
    }

}
```
