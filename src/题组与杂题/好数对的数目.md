# 好数对的数目

## [1512. 好数对的数目](https://leetcode.cn/problems/number-of-good-pairs/)

> - ***Question***
>   - 给你一个整数数组 `nums` 。
>   - 如果一组数字 `(i, j)` 满足 `nums[i] == nums[j]` 且 `i < j` ，就可以认为这是一组好数对。
>   - 返回好数对的数目。
>   - ***tips:***
>     - `1 <= nums.length <= 100`
>     - `1 <= nums[i] <= 100`

## Java

> - ***暴力***

```java
class Solution {
    
    public int numIdenticalPairs(int[] nums) {
        int ans = 0;
        for (int i = 0; i < nums.length; ++i) {
            for (int j = i + 1; j < nums.length; ++j) {
                if (nums[i] == nums[j]) {
                    ++ans;
                }
            }
        }
        return ans;
    }

}
```
